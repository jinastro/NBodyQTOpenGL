# README

利用 C++ QT OpenGL写的 N体模拟程序， 可以使用键盘鼠标移动视角

| 键位         | 功能           |
| ------------ | -------------- |
| W            | 向上移动摄像机 |
| A            | 向左移动摄像机 |
| S            | 向下移动摄像机 |
| D            | 向右移动摄像机 |
| Q            | 向前移动摄像机 |
| E            | 向后移动摄像机 |
| 鼠标左键拖动 | 旋转视角       |

引力公式，

 $G\frac{m_1m_2}{(r+r_{min})^2}$
